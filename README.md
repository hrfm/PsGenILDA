# PSGenILDA

### 1. Setup adobe-photoshop/generator-core

https://github.com/adobe-photoshop/generator-core

### 2. Placement

||||
|:--|:--|:--|
|adobe-photoshop|generator-core|(git-repos)|

### 3. npm install

    cd PSGenILDA
    npm install

### 4. Run Photoshop

### 5. Run PSGenILDA

    cd adobe-photoshop/generator-core
